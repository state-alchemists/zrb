## 2.49.4 (July 8, 2026)

- **Re-add `mistral` extra** (`pyproject.toml`): Restores the `mistral` optional dependency (`mistralai>=2.0.0,!=2.4.6`) that was emergency-removed after `mistralai==2.4.6` was quarantined on PyPI for a supply-chain attack. Pinned with the same exclusion pydantic-ai-slim's own `mistral` extra uses.
- **Install scripts: offer `mistral` in the all-extras install** (`install.sh`, `install.ps1`): `mistral` is now included alongside the other LLM provider SDKs when a user opts into installing all optional dependencies.
