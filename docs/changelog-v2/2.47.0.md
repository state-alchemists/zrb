## 2.47.0 (July 1, 2026)

- **Feature: PDF text extraction in attachments** (`src/zrb/llm/util/attachment.py`, `src/zrb/llm/util/pdf.py`): Extracted `extract_pdf_text()` into a shared utility (`pdf.py`) consumed by both the `read_file` tool and the attachment pipeline. Incoming PDF attachments are now parsed as text instead of raw binary, so the LLM can read them directly. Falls back to binary attachment when text extraction fails.

- **Refactor: `HasActivityTracking` protocol** (`src/zrb/llm/agent/activity.py`, `src/zrb/llm/tool/delegate.py`): Formal protocol interface for activity-tracking UIs, replacing ad-hoc `hasattr` checks with `isinstance` for better type safety and discoverability.

- **Fix: Termux detection for proot-distro** (`src/zrb/config/helper.py`): Also check `ANDROID_ROOT` (set to `/system` on all Android devices) so proot-based Linux distros running inside Termux are recognized as Termux environments.
