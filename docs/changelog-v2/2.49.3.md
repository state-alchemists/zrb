## 2.49.3 (July 8, 2026)

- **Install scripts: migrate from `pip`/`local-venv` to `pipx`** (`install.sh`, `install.ps1`): Rewritten to install `zrb` via `pipx` instead of `pip` into a local virtualenv. Adds automatic cleanup of legacy `~/.local-venv`, shell autocomplete registration, optional LSP server installation, and `-y`/`--yes` flag for non-interactive use. Windows installer now handles Python detection, version validation, and `pipx` bootstrap via winget/Scoop/pip.
- **Dependencies: `black` and `isort` moved to `[python]` extra** (`pyproject.toml`): These formatters are no longer installed by default. Run `pip install "zrb[python]"` or `pipx inject zrb "zrb[python]"` to get them.
- **`format-code` task: converted to `@make_task` with friendlier errors** (`src/zrb/builtin/python.py`): Replaced `CmdTask` with a `@make_task`-wrapped function that checks for `isort`/`black` before running and provides a helpful `[SYSTEM SUGGESTION]` when they're missing (covering both `pipx inject` and `pip install`).
- **Use `CFG.ROOT_GROUP_NAME` instead of hardcoded `"zrb"`** (`src/zrb/runner/cli.py`, `src/zrb/runner/chat/chat_api_route.py`, `src/zrb/llm/common_tools.py`, `src/zrb/llm/tool/zrb_task.py`): Tool group names, error messages, and server descriptions now respect the configured root group name.
- **Dockerfile: pinned `poetry install` to `--without dev`** (`Dockerfile`): Minor clarification тАФ the existing behavior was already `--without dev`, now explicit in a comment.
