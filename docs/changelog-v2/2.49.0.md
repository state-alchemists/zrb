## 2.49.0 (July 4, 2026)

- **Feature: deferred loading for rarely-used LLM tools** (`src/zrb/builtin/llm/chat.py`, `src/zrb/llm/common_tools.py`, `src/zrb/llm/agent/common.py`, `src/zrb/llm/agent/subagent/manager/manager.py`): Delegate tools, `analyze_code`/`analyze_file`, worktree tools, LSP tools, plan-mode tools, skill activation, process monitoring, and MCP toolsets are now wrapped with pydantic-ai's `defer_loading=True`, hiding their schemas from the model's context on every turn until it searches for them by name — the Tool Usage Guide still tells the model they exist, so nothing is lost except the standing token cost. `_wrap_tool` now threads `defer_loading` through when re-wrapping a `Tool`, and `SubAgentManager.add_tool`/`append_tool` accept `Tool` instances (not just plain callables) so deferred tools can be registered directly.
