## 2.46.0 (June 30, 2026)

- **Feature: sub-agent activity panel** (`src/zrb/llm/agent/activity.py`, `src/zrb/llm/app/layout.py`, `src/zrb/llm/tool/delegate.py`, `src/zrb/llm/ui/default/output_mixin.py`, `src/zrb/llm/ui/default/ui.py`): Live panel in the TUI showing all currently-running sub-agents with their ordinal number, task summary, and latest output line. Collapses to zero height when idle. Sub-agents are now labelled `[name #1]`, `[name #2]`, … (ordinal resets when the batch drains) instead of the previous opaque random-id suffixes, making the panel the legend for output prefixes.

- **Feature: PDF reading in `read_file`** (`src/zrb/llm/tool/file_read.py`): The LLM `read_file` tool now extracts text from PDF files via `pdfplumber` (lazy import). Supports the same `start_line`/`end_line` range and size-cap/truncation semantics as plain text files. Returns a clear error for scanned/image-only PDFs with no text layer.
