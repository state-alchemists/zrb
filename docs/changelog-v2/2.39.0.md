## 2.39.0 (June 21, 2026)

- **Feature: fully configurable LLM UI styles** (`src/zrb/config/mixins/llm_ui_styles.py`, `src/zrb/llm/app/style.py`, `src/zrb/llm/ui/default/output_mixin.py`): the `AskUserQuestion` choice-widget backgrounds (`ZRB_LLM_UI_STYLE_CHOICE_BG`, `ZRB_LLM_UI_STYLE_CHOICE_SELECTED_BG`), the five status-bar mode-badge styles (`ZRB_LLM_UI_STYLE_MODE_NORMAL/ACCEPT_EDITS/PLAN/YOLO/CUSTOM`), and the five info-bar yolo/plan indicator colors (`ZRB_LLM_UI_STYLE_INFO_YOLO_ON/PARTIAL/OFF`, `ZRB_LLM_UI_STYLE_INFO_PLAN_ON/OFF`) are now `EnvField`-backed — all accept prompt_toolkit style strings and default to the previously hardcoded values. The module-level `_MODE_STATUS_STYLE` dict in `output_mixin.py` was replaced with a lazy `_get_mode_status_style()` function that reads from `CFG` on every call, so env-var changes take effect without a restart. See [TUI Color Styles](../configuration/llm-config.md#20-tui-color-styles).

- **Fix: Termux keybinding compatibility** (`src/zrb/llm/ui/default/keybindings_mixin.py`, `src/zrb/llm/app/keybinding.py`, `src/zrb/llm/app/layout.py`, `src/zrb/llm/ui/base/commands_mixin.py`, ADR-0075): on Termux, `Tab` and `Shift+Tab` both produce the same byte (`0x09`) and cannot be distinguished. The focus-toggle binding was moved from `F6` (mapped to system actions on Termux) to `Ctrl+K` (overrides only Emacs-mode kill-line, which is less critical than focus traversal). `Tab` now also cycles modes alongside `Shift+Tab` — both route to the same handler, gated by `~has_completions` so a completion menu keeps Tab for next-completion. The `/help` shortcut list reflects both changes.

- **Security: dependency bumps** (`pyproject.toml`): `pydantic-settings>=2.14.2` (GHSA-4xgf-cpjx-pc3j — symlink traversal outside `secrets_dir`), `langsmith>=0.8.18` (GHSA-f4xh-w4cj-qxq8 — `TracingMiddleware` arbitrary server-side file read).

- **Documentation: theme examples** (`examples/themes/`): three shell scripts (`zrb-theme-dark.sh`, `zrb-theme-light.sh`, `zrb-theme-high-contrast.sh`) that set all `ZRB_LLM_UI_STYLE_*` env vars at once. Each defines a shell function for mid-session switching. Documented in [TUI Color Styles → Theme Examples](../configuration/llm-config.md#theme-examples).
