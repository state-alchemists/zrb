## 2.49.5 (July 8, 2026)

- **`zrb python format` works under pipx installs** (`src/zrb/builtin/python.py`): tool detection now uses `importlib.util.find_spec` instead of `shutil.which`, and `isort`/`black` run via `sys.executable -m` — so the formatters resolve from zrb's own venv even when their binaries are not on PATH (the default with pipx). The missing-tools suggestion now recommends `pipx inject zrb black isort --include-apps`.
- **Install scripts expose `black`/`isort` on PATH** (`install.sh`, `install.ps1`): when the `python` extra is selected, the installers re-inject `black` and `isort` with `--include-apps` so their binaries land in `~/.local/bin` (pipx only exposes the main package's entry points). Failure is non-fatal and prints the manual command.
