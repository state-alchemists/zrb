## 2.44.0 (June 27, 2026)

- **Feature: voice dictation via `/voice` command with terminal-safe push-to-talk** (`src/zrb/llm/voice/`, `src/zrb/config/mixins/llm_voice.py`, `src/zrb/config/mixins/llm_ui_commands.py`, `src/zrb/config/mixins/llm_ui.py`, `src/zrb/llm/ui/base/commands_mixin.py`, `src/zrb/llm/ui/base/ui.py`, `src/zrb/llm/ui/default/keybindings_mixin.py`, `src/zrb/llm/ui/default/output_mixin.py`, `src/zrb/llm/ui/default/ui.py`, `src/zrb/llm/task/chat/task.py`, `src/zrb/llm/task/chat/exec_mixin.py`, `src/zrb/llm/task/chat/runner_mixin.py`, `docs/adr/adr-0081.md`): Opt-in voice dictation for the chat TUI. Enable with `ZRB_LLM_VOICE_ENABLED=true`, then type `/voice` to toggle push-to-talk mode. When active, the first press of the spacebar (configurable via `ZRB_LLM_VOICE_PUSH_TO_TALK_KEY`) starts recording and a second press stops it and exits voice mode; a 300ms debounce filters OS key-repeat (terminals can't detect key-release). Transcribed text inserts at the cursor position, enabling hybrid typing+voice input. Supports four STT backends via `ZRB_LLM_VOICE_MODE`: `vosk` (offline, default — model auto-downloads to `~/.cache/vosk/` on first use; the download is cancellable with `/q` or Ctrl+C), `openai` (Whisper API), `google` (Gemini STT), or `multimodal` (uses `ZRB_LLM_MULTIMODAL_MODEL`). All audio dependencies are lazy-imported — no startup cost when voice is disabled. See ADR-0081.
