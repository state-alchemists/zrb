## 2.43.0 (June 25, 2026)

- **Feature: add `zrb git changelog generate` command** (`src/zrb/builtin/changelog.py`, `src/zrb/builtin/changelog_template.md`, `src/zrb/builtin/group.py`, `src/zrb/builtin/__init__.py`): A new built-in task that generates one changelog file per matching git tag using the LLM. It enumerates tags matching a configurable regex (default matches `v1.2.3`, `1.2.3`, pre-release suffixes, etc.), and for each tag without an existing file summarizes the commit range since the previous tag into a templated markdown file, so reruns are incremental. The LLM gets a read-only `git` tool restricted to `diff`, `log`, `show`, `shortlog`, and `tag` to inspect diffs where the commit log is insufficient. Inputs (`dir`, `pattern`, `sort`, `template`) are configurable.

- **Improvement: migrate foundation, internet-search, and LLM-UI config knobs to `EnvField`** (`src/zrb/config/env_field.py`, `src/zrb/config/mixins/foundation.py`, `src/zrb/config/mixins/internet_search.py`, `src/zrb/config/mixins/llm_ui_styles.py`): ~20 hand-written `@property` getter/setters (`ENV_PREFIX`, `SHELL`, `EDITOR`, `INIT_FILE_NAME`, `INIT_SCRIPTS`, `INIT_MODULES`, `VERSION`, `LOGGING_LEVEL`, `BRAVE_API_KEY`, `SERPAPI_KEY`, `LLM_ASSISTANT_NAME`, and more) are now declarative `EnvField` descriptors. A new `no_prefix=True` option reads/writes the bare env name (for keys outside the `ZRB_` namespace like `BRAVE_API_KEY`/`SERPAPI_KEY`, and bootstrap keys `_ZRB_ENV_PREFIX`/`_ZRB_CUSTOM_VERSION`); irregular knobs are expressed via existing `transform`/`default_factory`/`aliases`. Flat access (`CFG.INIT_FILE_NAME`) is unchanged. **Side effect:** `zrb config explain` only lists `EnvField`-backed knobs, so these variables (e.g. `ZRB_INIT_FILE_NAME`, `BRAVE_API_KEY`) now appear in its output where they were previously omitted.

- **Improvement: `ZRB_INIT_MODULES` now accepts a comma-separated list** (`src/zrb/config/env_field.py`, `docs/configuration/env-vars.md`): Module names are parsed with a new `comma_or_colon_list` helper that accepts both `,` and `:` delimiters, so the colon-separated form written before this change keeps working.

- **Fix: output pane scrolling in the LLM chat UI** (`src/zrb/llm/app/layout.py`, `src/zrb/llm/ui/default/output_mixin.py`): The mouse wheel now moves the output cursor instead of the viewport, which actually scrolls the cursor-pinned output window and pauses auto-follow when you scroll up; scrolling back to the last line resumes following the stream. Works regardless of which pane is focused, so the streamed output/thinking can be read mid-stream without first focusing the output pane (Ctrl+K).

- **Fix: copying from the output pane now strips ANSI codes** (`src/zrb/llm/ui/default/keybindings_mixin.py`): The output buffer holds raw style escape codes (e.g. muted tool-call detail); the copy keybinding now runs the selection through `remove_style` so the clipboard receives plain text.

- **Improvement: `/load` lists most-recently-used sessions first** (`src/zrb/llm/history_manager/file_history_manager.py`): Conversation search now sorts by fuzzy score then modification time (newest first). With an empty keyword every score ties, so recent sessions surface at the top.
