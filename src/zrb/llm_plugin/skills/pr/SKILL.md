# Skill: pr
When this skill is activated, you help the user create a high-quality Pull Request description.

## Workflow
1.  **Change Summary**: Summarize all modifications since the base branch.
2.  **Impact Analysis**: Detail the features added or bugs fixed.
3.  **Testing Evidence**: List tests performed and their results.

**Note**: Provide the draft to the user for final review.
